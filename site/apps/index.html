Apps and projects that I have made.

  • Handmade Network website: Made together with Asaf Gartner, the website for the Handmade Network acts as a hub for the community. The website is completely custom and supports forums, blogs, feeds of project activity, a podcast, a robust custom Discord integration, and more. Now open source and developed with the community.
  • Regular Express: A visual regular expression editor, allowing you to create and edit complex regular expressions without touching the syntax. Created for the web in C and WebAssembly. Currently in alpha.
  • Database Explorer: Explore a relational database fluidly and naturally, using a system inspired by node graphs from film and game software. Created with Ejekaflex for the 2021 Wheel Reinvention Jam. Details and video here.
  • Name My Church: A tongue-in-cheek church name generator.
  • Beach Ball Valley: A whimsical VR experience with too many beach balls. Created with Paul Eckhardt.
  • Adrift: A bullet hell where you can't move. Made in Godot for the 2018 GMTK Jam.
  • Territoris: A puzzle game where your kingdom behaves like a slide puzzle. Made in Unity with Joseph Merboth for Ludum Dare 35.
  • Boggler: A speedy Boggle solver.
  • Sandbox: A modern replacement for Eric Gossett's Sandbox applet. Used by Bethel University in their Discrete Math courses - at least when I was a student!
